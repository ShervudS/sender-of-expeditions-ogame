# sender-of-expeditions-ogame

## For run:

`docker-compose up`

---

### Example `.env` file

- API_PORT - port with express;
- TG_BOT - tg bot token;;
- CHAT_ID - chat id in tg;
- TOKEN - token on serverless;
- O_GAME_SITE_URL - ogame site url;
- USER_MAIL - email ogame user;
- USER_PASSWORD - pass ogame user;
